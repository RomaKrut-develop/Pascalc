----------------------------------------Pascalc Manual------------------------------------------------------

INFO:
	Pascalc is open-source project, maded and programed on 'Turbo Pascal 7.0'.
	Developer is CD-ROM SOFT (RomanBraginetz).
	
REQUIREMENTS:
	
	GPU - Supported CGA, EGA, VGA Videocards. (16+ colors);
	RAM - 1 or 2 MB;
	Drivers - EGAVGA.BGI or others.
	
EXTRA:
	
	There is two versions of Pascalc.

	In 'Hardcore' folder, pascal version without mouse support. Only keyboard

	In 'Softcore' folder, pascal version with mouse support
